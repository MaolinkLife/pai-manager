## 🧠 1. Введение

**Z-Waif** — открытая платформа для локального развёртывания ИИ-компаньона.  
Она объединяет диалоги, синтез речи, визуализацию, поведенческую логику и глубокую кастомизацию.

### Компоненты

- **Ollama** — основной локальный LLM-движок для диалогов  
- **OpenRouter и внешние API** — облачные провайдеры для расширенных возможностей  
- **Z-Waif** — интерфейс и логика: TTS, STT, RVC, VTube Studio, Lorebook, память и др.  
- **RVC / Voice Changer** — синтез и персонализация голоса  
- **VTube Studio** — визуальный аватар  
- **Whisper / STT** — распознавание речи  

_Цель: объединить всё в одном месте для удобства разработки и использования._

## 🔧 3. Что реализовано (v0.1–v0.4)

### ✅ 3.1. Полное управление через UI
- **Полностью переработанный интерфейс** — все настройки доступны из UI  
- **Визуальные редакторы** для памяти, лорбуков и системных промптов  
- **Поддержка тем** — светлая/тёмная с сохранением предпочтений  
- **Расширенная панель диагностики** с визуальным логом  

### ✅ 3.2. Мультиязычность и русская локализация
- **Переключение интерфейса EN/RU**  
- **Whisper адаптирован под русский язык**  
- **Улучшен STT для русской речи**  
- **Полная поддержка UTF-8** во всех модулях  
- **JSON-файлы перевода** для системных сообщений (частично)  

### ✅ 3.3. Голосовой интерфейс
- **Выбор голоса** (мужской/женский) из UI при отключённом RVC  
- **VAD (Voice Activity Detection)** с ключевыми словами активации  
- **Мульти-провайдерный TTS:** ElevenLabs → gTTS → pyttsx3 (оффлайн)  
- **Контроль состояния голоса** (прослушивание / ожидание / речь)  

### ✅ 3.4. Память и интеллект
- **Lorebook в базе данных** — автозагрузка устарела, теперь интегрирован в БД  
- **Гибридный поиск** — embeddings + вектор + ключевые слова  
- **MemoryModule** с RAG-поиском и метаданными  
- **Инициатива LLM** и цепочка провайдеров  

### ✅ 3.5. Платформа и архитектура
- **Модульная архитектура** (voice, vision, memory, LLM)  
- **SQLite-база данных** с хранением reasoning и медиа  
- **Единая система конфигураций** с синхронизацией UI↔backend  

### ✅ 3.6. Провайдеры LLM
- **Ollama** — основной локальный движок (заменил Oobabooga)  
- **OpenRouter и внешние API** — облачные провайдеры  
- **Автоматическое переключение** между провайдерами при сбое  
- **Унифицированный API** для всех источников

## ❓ 5. FAQ

**Q1:** Нужно ли вручную редактировать конфиги?  
**A1:** **Нет!** Все настройки доступны через удобный интерфейс.  

**Q2:** Какие LLM поддерживаются?  
**A2:** **Ollama (локально)** + **OpenRouter и внешние API** (облако).  

**Q3:** Есть ли поддержка русского языка?  
**A3:** **Полная** — интерфейс, STT, TTS, память.  

**Q4:** Может ли работать без интернета?  
**A4:** **Да.** Ollama работает полностью офлайн.  

**Q5:** Какие минимальные требования к ПК?  
**A5:** Зависит от модели Ollama. Минимум 8 ГБ ОЗУ, рекомендуется 16+.

## ⚙️ 6. Технические особенности

- **Zero-Code управление** — всё через UI, без правки файлов  
- **Гибридная память** — поиск по векторам + ключевым словам + RAG  
- **Модульная архитектура** — voice / vision / memory работают независимо  
- **Поддержка тем** — светлая/тёмная с автосохранением  
- **Фокус на Ollama** — оптимизация под современные локальные модели

## 🚀 7. Заключение

**Z-Waif 0.4 — полностью самостоятельная система управления ИИ-компаньоном.**  

Больше не нужно редактировать конфиги, копаться в коде или искать параметры в файлах.  
Всё — от выбора голоса до настройки памяти — доступно в интуитивном интерфейсе.  

**Ключевые изменения:**  
- **Полный переход на Ollama** вместо Oobabooga  
- **Интеграция OpenRouter** и внешних API  
- **Упрощённая установка** — меньше зависимостей  

Система видит, слышит, запоминает и говорит по-русски, а модульная архитектура позволяет легко расширять функциональность.  
**Режим Jarvis становится реальностью!**
